# Инструкция по запуску приложения

## Быстрый старт

### 1. Настройка PostgreSQL
1. Установите PostgreSQL (если не установлен)
2. Создайте базу данных `construction_materials_db`
3. Выполните SQL скрипт из файла `database_schema.sql`

### 2. Настройка подключения
Откройте файл `appsettings.json` и измените параметры подключения:

```json
{
  "DatabaseSettings": {
    "Host": "localhost",
    "Port": 5432,
    "Database": "construction_materials_db",
    "Username": "postgres",
    "Password": "ваш_пароль_postgres"
  }
}
```

**Замените `"ваш_пароль_postgres"` на ваш реальный пароль от PostgreSQL!**

### 3. Запуск приложения
```bash
dotnet build
dotnet run
```

## Основные функции

- **Стройматериалы**: управление каталогом материалов
- **Заказчики**: ведение базы клиентов  
- **Отгрузки**: создание и учет отгрузок
- **Отчеты**: формирование отчетов с экспортом в CSV

## Тестовые данные

После выполнения `database_schema.sql` в базе будут созданы тестовые материалы и заказчики для демонстрации функциональности.
